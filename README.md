# GPU-Compute-Capability

An application for querying the computing power of each gpu released by NVIDIA.

## Install

```bash
pip install -r requirements.txt
```

## Usage

```bash
git clone https://github.com/XavierJiezou/GPU-Compute-Capability.git
```
